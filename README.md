## How to Use

1. Clone this repository to your desired location.
   ```bash
   git clone https://github.com/rgxcp/anarchi.git
   ```
2. Change your directory into project.
   ```bash
   cd anarchi
   ```
3. Run the script.
   ```bash
   bash anarchi.sh
   ```
4. Do-what-you-want-with-it!
